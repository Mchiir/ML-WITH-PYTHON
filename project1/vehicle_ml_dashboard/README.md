# Vehicle Machine Learning Dashboard

## Overview

This is a Django-based web application designed to perform exploratory data analysis (EDA) on vehicle sales data. This system presents structured statistical summaries and grouped analytical tables through a clean and navigable web interface.

This project demonstrates how data processing, aggregation, and tabular analysis can be integrated into a web framework for analytical reporting purposes.

---

## Project Objectives

- Perform one-way frequency analysis on categorical variables
- Preview structured dataset samples
- Compute aggregated financial metrics
- Generate cross-tabulated summaries
- Produce pivot table analyses
- Present results in a professional web dashboard

---

## Core Functionalities

### Frequency Table

Generates one-way frequency counts for categorical attributes such as manufacturers.

### Dataset Preview

Displays a subset of the dataset to provide structural insight into available features.

### Aggregated Data

Computes grouped totals and derived metrics such as profit based on selling and wholesale prices.

### Crossed Table

Produces multi-dimensional summaries using cross-tabulation techniques.

### Pivot Table

Generates advanced grouped summaries with multiple aggregation functions.

---

## Technologies Used

### Django

Used as the web framework to handle routing, request processing, and template rendering.

### Pandas

Used for data loading, transformation, grouping, aggregation, and statistical analysis.

### Plotly

Included for potential interactive data visualization capabilities.

### Bootstrap

Used for responsive layout design and professional dashboard styling.

### Python

Serves as the primary programming language for backend logic and data processing.

---

## Architectural Structure

- Modular app-based Django structure
- Separation of data processing logic into a dedicated dashboard module
- Template-based rendering for dynamic HTML generation
- Static dataset integration for analytical evaluation

---

## Educational Value

This project illustrates the integration of data science techniques within a web development environment. It reinforces concepts such as groupby operations, pivot tables, cross-tabulation, and structured data reporting in a full-stack context.

This dashboard serves as a foundational model for building scalable analytical systems and machine learning reporting platforms.
