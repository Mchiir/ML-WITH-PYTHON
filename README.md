# PYTHON-WORKSPACE

> Commands for adding FarmManagementSystem/.git submodule after defining child .git in FarmManagementSystem DIR

```bash
F:\GIT-Push\PYTHON-WORKSPACE\FarmManagementSystem> cd F:\GIT-Push\PYTHON-WORKSPACE
git submodule add https://github.com/Mchiir/FarmManagementSystem.git FarmManagementSystem/
git add .gitmodules FarmManagementSystem/
git commit -m "Added FarmerApp as a git submodule"
git push origin main
```
